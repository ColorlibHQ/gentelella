Questions

Frequently asked

Is it really free for commercial use? +
Yes. MIT licensed since day one. Use it for client work, side projects, your own SaaS — no attribution required (though we always appreciate a star on GitHub).
Do I need a backend? +
No. Gentelella v4 is a pure frontend template — static HTML/CSS/JS. Wire it up to whatever stack you like (Rails, Django, Express, Laravel, Supabase, anything).
Can I use it with React/Vue/Svelte? +
The SCSS design system is framework-agnostic — drop the compiled CSS into any project. The HTML pages and shell rendering are vanilla JS, so port what you need.
How do I customize the colors? +
Every color lives in _tokens.scss as a CSS custom property. Change --primary and every chart, button, badge, and link updates.
What changed in v4? +
Bootstrap and jQuery are gone. The design system is custom SCSS. ECharts replaced Chart.js. DataTables replaced jQuery DataTables. node_modules dropped from ~600 MB to 138 MB. See the changelog for the full diff.
Where can I get help? +
Open an issue on GitHub or read the help center. The repo also has a CONTRIBUTING.md if you want to send a PR.

Free for personal & commercial use

MIT licensed. Use it for client work, side projects, anything. The dashboard you saw above is the live demo — every page works.
