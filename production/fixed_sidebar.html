Fixed sidebar

Default behavior in v4. The sidebar uses position: fixed by default and scrolls independently from the main content. Try scrolling — the sidebar stays in place.
Long content for scroll testing

The v4 layout is built around a fixed sidebar (.sidebar { position: fixed }) and a fixed topbar (.topbar { position: fixed }). The main content scrolls under both.

This page exists as a smoke test — load it and scroll. The sidebar should stay anchored to the left edge regardless of how far down you go, and so should the topbar at the top.
